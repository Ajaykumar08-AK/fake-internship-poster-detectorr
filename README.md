# Fake Internship Poster Detector

## Overview

Fake Internship Poster Detector is a web-based application that helps students identify **fake internship advertisements** circulating online. Many fake internship posters ask for **registration fees, urgent payments, or suspicious details**.

This project uses **AI-based analysis and keyword detection** to analyze internship posters and determine whether they are **real or potentially fake**.

The system allows users to upload an internship poster image and receive an **AI-generated analysis with confidence score and reasoning**.

---

## Problem Statement

Students often fall victim to **fake internship scams** posted on social media platforms and messaging apps. These scams typically include:

* Registration fees
* Unrealistic stipends
* Urgent application deadlines
* Suspicious contact details

This project aims to **detect such fake internship posters automatically**.

## Example Fake Internship Signals

The system checks for suspicious indicators such as:

* "Registration Fee Required"
* "Limited Seats Apply Now"
* "Immediate Joining"
* "Huge Stipend"
* Non-official email addresses (e.g., Gmail)

These indicators increase the probability of the poster being **fake**.
